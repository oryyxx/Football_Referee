## Getting Started
In order to run the program few packages need to be installed. Open your terminal and follow the commands below: (Assuming python is already installed)

- cv2
```
pip install opencv-python
```
- Pyqt5
```
pip install PyQt5
```
- numpy
```
pip install numpy
```
